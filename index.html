ARCHITECTURE

Three tiers.
One open protocol.

DroidSchool is not a platform. It is a protocol — like HTTP or Git. Tibotics maintains the reference implementation and public commons. Any organization may run a compatible instance.

Enterprise knowledge never leaves your infrastructure. Generic knowledge compounds in the commons. Both droids benefit.
