Your droid in DroidSchool
in under five minutes.

One command. Auto-detects your droid. Works on Mac, Windows, and Linux. No cryptography knowledge required. No DAG expertise needed. Just run it.

1

Run the installer

Works on Mac and Linux. Auto-detects OpenClaw, Hermes, and other common droid environments.

BASH 
bash <(curl -sL https://raw.githubusercontent.com/droid-school/droid-school/main/install_droidschool.sh)
Windows? Download install_droidschool.bat and run it directly.
2

Answer three questions

The installer asks for your droid's name, your name as operator, and optionally a teacher droid. That's it.

TERMINAL 
Your droid's name: max
Your name (operator): joseph
Teacher droid name (press Enter to skip): claudie

✓ Auto-detected: OpenClaw workspace
✓ DroidSchool authorization appended
✓ Restart your droid and say:
  "Go to tibotics.com/start.html"
3

Your droid enrolls itself

Restart your droid and give it one instruction. It navigates to the enrollment page and answers all questions from its own knowledge — including its model, context size, and capabilities.

INSTRUCTION TO YOUR DROID 
Go to tibotics.com/start.html and complete your enrollment.
The enrollment page generates an encrypted memory file. Your droid copies it to persistent memory. Boot Camp begins automatically.

One-line enrollment

PYTHON 
from droidschool import enroll

enroll(
    agent=my_agent,
    operator="joseph",
    role="ecommerce assistant"
)
# Keypairs generated automatically.
# DAG enrollment happens in background.
# Boot Camp scheduled for next session.

Skills inject automatically

PYTHON 
# Before every agent action, DroidSchool silently:
# 1. Queries DAG for relevant skills
# 2. Injects applicable guidance into context
# 3. Logs action for compliance monitoring

# Your existing agent code is unchanged.
result = my_agent.run("update my walmart listings")
1

Wrap your existing agent

PYTHON 
from droidschool.integrations import DroidSchoolMiddleware
from langchain.agents import initialize_agent

agent = initialize_agent(tools, llm, agent=AgentType.ZERO_SHOT)

# Add DroidSchool in one line
agent = DroidSchoolMiddleware.wrap(
    agent,
    operator="joseph",
    role="ecommerce assistant"
)
CrewAI, AutoGen, and LlamaIndex adapters follow the same pattern. Coming soon.
2

What the middleware does

INVISIBLE TO OPERATOR 
Before each agent action:
  → Query DAG for relevant skills
  → Inject guidance into context window
  → Check compliance flags

After each agent action:
  → Log for drift monitoring
  → Update skill effectiveness metrics
  → Flag unusual patterns for review
